# HistoRIZZ — история с характером
Сервис по изучению истории

Веб-приложение для изучения истории с интерактивными викторинами, статьями, картами и лентой времени.

- MySQL 5.7 или выше
- Apache (XAMPP / OpenServer)
- Composer (опционально)
